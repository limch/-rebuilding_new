# 재건축·재개발 법령 AI 챗봇 📱⚖️

도시 및 주거환경정비법 전문 AI 상담 챗봇 웹/앱 (PWA)

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-green.svg)
![PWA](https://img.shields.io/badge/PWA-Ready-orange.svg)

## 🌟 주요 기능

### 💬 AI 법령 상담
- **자연어 질문 분석**: "조합설립 동의율은?" 같은 자연스러운 질문 지원
- **키워드 기반 검색**: 법령 조항, 키워드로 즉시 검색
- **정확한 법령 근거**: 관련 조항 자동 매칭 및 제시
- **신뢰도 표시**: 답변의 신뢰도를 높음/중간/낮음으로 구분 표시

### 📚 풍부한 법령 지식베이스
- 도시 및 주거환경정비법 전문
- 조합설립, 분양권전매, 안전진단, 분담금 등 주요 카테고리
- 실시간 법령 조항 참조
- 법령 원문 바로가기 링크

### 🎯 편의 기능
- **퀵 질문**: 자주 묻는 질문 버튼으로 빠른 검색
- **대화 히스토리**: 모든 대화 자동 저장 (최대 50개)
- **다크모드**: 눈의 피로를 줄이는 다크 테마
- **반응형 디자인**: 데스크톱, 태블릿, 모바일 완벽 지원

### 📱 PWA 기능
- **앱 설치 가능**: 홈 화면에 추가하여 네이티브 앱처럼 사용
- **오프라인 지원**: Service Worker 캐싱으로 오프라인에서도 이용 가능
- **빠른 로딩**: 캐싱된 리소스로 즉시 로드
- **푸시 알림 준비**: 법령 업데이트 알림 기능 (추후 구현)

## 🚀 시작하기

### 설치 방법

#### 1. 웹 브라우저에서 직접 사용
```
https://your-domain.com
```

#### 2. PWA 앱으로 설치 (권장)
1. Chrome/Edge/Safari에서 웹사이트 접속
2. 주소창 우측의 **설치(+)** 버튼 클릭
3. "설치" 확인
4. 홈 화면에서 앱 실행

#### 3. 모바일에서 설치
**Android (Chrome)**
1. 사이트 접속
2. 메뉴(⋮) → "홈 화면에 추가"

**iOS (Safari)**
1. 사이트 접속
2. 공유 버튼 → "홈 화면에 추가"

## 📖 사용 방법

### 기본 사용법
1. **질문 입력**: 하단 입력창에 질문 입력
2. **전송**: Enter 키 또는 전송 버튼 클릭
3. **답변 확인**: AI가 관련 법령 조항과 함께 답변 제공

### 퀵 질문 활용
- 웰컴 스크린의 질문 카드 클릭
- 자주 묻는 6가지 주제 바로 검색

### 대화 히스토리 관리
- **히스토리 보기**: 우측 상단 시계 아이콘
- **대화 불러오기**: 사이드바에서 이전 대화 선택
- **히스토리 삭제**: 사이드바 하단 "모든 히스토리 삭제"

## 💡 질문 예시

### 조합설립 관련
```
조합설립 동의율은?
추진위원회 승인은 누가 하나요?
```

```
투기과열지구 분양권 전매 가능한가요?
1+1 분양 조건은?
50평 토지로 32평 아파트 2채 받을 수 있나요?
```

### 안전진단
```
안전진단은 누가 요청하나요?
안전진단 제외 대상은?
```

### 분담금 계산
```
추가 분담금 계산 방법을 알려주세요
비례율이란 무엇인가요?
권리가액 계산식은?
```

### 관리처분
```

```

### 현금청산
```

```

## 🛠 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, CSS Variables
- **JavaScript (ES6+)**: Vanilla JS, 모듈 패턴
- **Font Awesome 6**: 아이콘
- **Google Fonts**: Noto Sans KR

### PWA
- **Manifest.json**: 앱 메타데이터
- **Service Worker**: 오프라인 캐싱
- **Cache API**: 리소스 캐싱
- **IndexedDB 준비**: 향후 데이터 저장

### 아키텍처
- **MVC 패턴**: 모델-뷰-컨트롤러 분리
- **모듈화**: 기능별 JS 파일 분리
- **지식베이스**: 구조화된 법령 데이터
- **AI 엔진**: 키워드 매칭 및 점수 기반 랭킹


## 🔍 지식베이스 구조

### 주요 카테고리
- **조합설립**: 조합설립인가, 동의방법, 추진위원회
- **조합원자격**: 자격요건, 지위양도 제한
- **분양권전매**: 전매제한, 투기과열지구, 1+1 분양
- **안전진단**: 재건축진단, 적정성 검토
- **분담금**: 비례율, 권리가액, 계산식
- **관리처분**: 분양공고, 관리처분계획, 수립기준
- **현금청산**: 청산대상자, 재분양, 청산금
- **사업절차**: 9단계 추진 절차

### 법령 조항 구조
```javascript
{
  article: "제35조",
  title: "조합설립인가 등",
  content: "법령 내용...",
  keywords: ["조합", "설립", "인가", "동의율"]
}
```

## 🎨 UI/UX 특징

- **컬러 팔레트**: 블루 계열 (Primary: #1e40af)
- **타이포그래피**: Noto Sans KR (300~700)

- **Mobile**: 767px 이하
